# Workflow: Manage Student Profiles

**Module:** edu
**Phase:** Anytime
**Owner Agent:** student-simulator

---

## Overview

Gestionar los perfiles de alumno disponibles para el simulador pedagógico.

## Steps

### Step 1: List Profiles
- **Agent:** student-simulator
- **Action:** Display current profiles with their characteristics

### Step 2: Action Selection
- **Options:**
  - **Add:** Create new profile based on academic literature (ERIC, ACM)
  - **Edit:** Modify existing profile parameters
  - **View:** Show detailed profile with cognitive characteristics
  - **Calibrate:** Adjust profile based on real survey data

### Step 3: Persist
- **Output:** Updated profile data in `_edu-memory/calibracion-simulador/`
